Overview:
This project uses Machine Learning to classify iris flowers into three species based on flower measurements. The model is trained using the famous Iris dataset available in Scikit-learn.

Technologies Used:
Python
Scikit-learn
NumPy
Pandas (Optional)

Features:
Loads Iris dataset
Trains Random Forest Classifier
Predicts flower species
Evaluates model accuracy
Generates classification report
Dataset Features
Sepal Length
Sepal Width
Petal Length
Petal Width

Project Workflow:
Iris Dataset
      ↓
Data Preprocessing
      ↓
Train-Test Split
      ↓
Model Training
      ↓
Prediction
      ↓
Accuracy Evaluation

How to Run:
pip install scikit-learn
python iris_classification.py

Output:
Species Prediction
Accuracy Score
Classification Report
